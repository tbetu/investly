# Investly

Investly is a mobile investment simulation and financial learning app built with **React Native** and **Expo**.
The app helps users learn basic investing concepts through interactive scenarios, portfolio tracking, and guided onboarding flows.

## Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* Expo Go app on your mobile device

### Installation

```bash
npm install
```

### Run the App

```bash
npx expo start
```

Scan the QR code with **Expo Go** (Android) or the **Camera app** (iOS).
